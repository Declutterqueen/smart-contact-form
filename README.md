# Smart Contact Form – UX Focused & Interactive

## Overview
This is an interactive **contact form** designed with **user experience (UX) and problem-solving** in mind.  
It provides **real-time guidance, visual feedback, and smooth animations** to make form submission **frustration-free**.  
The form is also **Netlify-ready**, so submissions can be collected without a backend.

---

## Features

- **Smart Helper Text**: Real-time messages guide the user while typing.
- **Shake Animation on Error**: Invalid fields shake to indicate an issue.
- **Submit Button Enable/Disable**: Button activates only when all fields are valid.
- **Success Pop-Up Animation**: Smooth animation confirms message submission.
- **Responsive Design**: Works on both desktop and mobile screens.
- **No Backend Needed**: Fully compatible with Netlify form handling.
